# N_m3u8DL-RE-GUI

N_m3u8DL-RE 的第三方图形界面（GUI）。通过可视化的方式拼接命令行参数，并调用官方的
`N_m3u8DL-RE` 可执行文件完成 DASH / HLS / MSS 流媒体的下载、解密与合并。

本项目使用 **Rust** 语言 + **[iced](https://github.com/iced-rs/iced)** 0.14 图形框架编写，
原生、轻量、跨平台（当前主要面向 Windows 验证）。它只是一个“参数生成器 + 启动器”，
真正的下载工作由上游 `N_m3u8DL-RE` 完成。

---

## 功能特性

GUI 将 `N_m3u8DL-RE` 的主要命令行选项按分类整理到多个标签页中：

| 标签页 | 主要内容 |
| --- | --- |
| **基本** | 可执行文件路径、下载地址/文件、保存目录/文件名/命名模板、临时目录、日志文件路径、BaseURL、线程数、重试次数、超时、请求头、代理（系统/不使用/自定义）、限速、并发下载、界面语言、ffmpeg 路径、解密引擎与工具路径 |
| **流选择** | 自动选择最佳轨道、按表达式选择/去除视频/音频/字幕（`-sv`/`-sa`/`-ss`/`-dv`/`-da`/`-ds`）、仅选字幕、字幕格式（SRT/VTT） |
| **解密** | 解密密钥 `--key`、密钥文件 `--key-text-file`、自定义 HLS 加密方式、HLS 解密 KEY/IV、实时解密 MP4 分片 |
| **直播** | 点播方式录制、实时合并、保留分片、管道混流、VTT 字幕修正、录制时长限制、列表刷新间隔、首次获取分片数等 |
| **高级** | URL 参数附加、分片数量检测、输出 meta json、日志级别、urlprocessor 参数、分片范围、延时开始、广告关键字、混流参数 `-M`、外部媒体导入、二进制合并、各种开关 |
| **日志** | 实时运行日志、复制命令、打开输出文件夹、开始下载按钮 |

其它特性：

- **命令预览**：底部实时显示拼装好的命令行，可一键复制。
- **外部控制台模式**：勾选后不捕获日志，而是像旧版 SimpleG 那样弹出独立控制台窗口运行。
- **配置持久化**：记住可执行文件路径、保存目录、代理地址、请求头、外部控制台偏好等。
- **自动探测**：未手动指定时，会按顺序在「用户指定路径 → 与 GUI 同目录 → 系统 PATH」中查找 `N_m3u8DL-RE` 可执行文件。

## 环境要求

1. **Rust 工具链**：edition 2024，建议使用 **Rust 1.85 或更高版本**（通过
   [rustup](https://rustup.rs/) 安装）。
2. **N_m3u8DL-RE 可执行文件**：本 GUI 本身不包含下载器，需自行准备官方构建的
   `N_m3u8DL-RE`（Windows 下为 `N_m3u8DL-RE.exe`）。
   - 可放在与 GUI 可执行文件相同的目录，或加入系统 `PATH`，GUI 会自动探测；
   - 也可在「基本」页手动指定其完整路径。

## 构建

```bash
# 调试构建（开发用）
cargo build

# 发布构建（推荐，体积小、运行快）
cargo build --release
```

构建产物：

- 调试：`target/debug/n_m3u8dl-re-gui.exe`
- 发布：`target/release/n_m3u8dl-re-gui.exe`

Windows 下首次构建会编译 `iced` / `wgpu` 等依赖，耗时较长，请耐心等待。

---

## 运行

```bash
# 直接运行（调试构建）
cargo run

# 或运行已构建的可执行文件
./target/release/n_m3u8dl-re-gui.exe
```

首次使用时，请在「基本」页填写 `N_m3u8DL-RE.exe` 的路径与下载地址，然后点击
「开始下载」。运行过程中可切换到「日志」页查看输出。

## 使用说明

1. **指定下载器**：在「基本」页设置 `N_m3u8DL-RE.exe` 路径（或在 GUI 同目录 / PATH 中放置它）。
2. **填写下载地址**：支持在线 URL，也支持本地 `.m3u8` / `.mpd` 文件路径。
3. **按需配置选项**：在对应标签页勾选/填写需要的参数；不修改的选项不会出现在命令行中
   （仅当与 `N_m3u8DL-RE` 默认值不同时才会输出）。
4. **命令预览**：底部文本框始终显示最终命令行，确认无误后可点「复制命令」手动执行。
5. **开始下载**：点击「开始下载」；GUI 会在内部捕获日志并展示在「日志」页。若勾选了
   「使用外部控制台窗口」，则会弹出独立控制台运行（不捕获日志）。

> 各选项对应的 `N_m3u8DL-RE` 命令行参数与含义，请以 `N_m3u8DL-RE` 官方文档为准。

## 配置存储

GUI 会记住少量用户偏好（可执行文件路径、保存目录、代理地址、请求头、外部控制台开关），
保存为 JSON 配置文件：

- Windows：`%APPDATA%\n_m3u8dl-re-gui\config\config.json`
- Linux / macOS：位于对应平台的应用配置目录下

删除该文件即可恢复默认设置。

## 项目结构

```
src/
├── main.rs      # 程序入口，iced 应用装配与字体加载
├── app.rs       # 应用状态 App、消息枚举 Message、参数拼装辅助
├── args.rs      # 将 App 状态转换为 N_m3u8DL-RE 命令行参数
├── config.rs    # 配置的加载与持久化（JSON）
├── detect.rs    # 定位 N_m3u8DL-RE 可执行文件
├── runner.rs    # 启动子进程、捕获输出、日志订阅
└── ui.rs        # 各标签页的界面布局
```
